# 3. **Helper Functions: Data Loading and Saving**

### a. `load_data()` — Load expense data from file or create empty with correct types

```python
def load_data():
    if os.path.exists(EXPENSE_FILE):
        df = pd.read_csv(EXPENSE_FILE)
        # Parse Date column with expected string format
        df["Date"] = pd.to_datetime(df["Date"], format="%d-%m-%Y %I:%M %p", errors="coerce")
        df = df.dropna(subset=["Date"])  # Remove rows where date conversion failed
        return df
    
    # If file doesn't exist, create empty DataFrame with types specified (important for .dt accessor)
    df = pd.DataFrame(columns=["Date", "Category", "Amount", "Description"])
    df["Date"] = pd.to_datetime(df["Date"])  # Set as datetime dtype, even if empty
    df["Amount"] = df["Amount"].astype(float)
    return df
```

- Tries to read the CSV file.
- Converts "Date" column from string to datetime with a specific format.
- Drops rows that fail date parsing.
- If no file exists, creates an _empty_ DataFrame with the right column data types — this prevents errors when accessing date functions later on `.dt`.

---

### b. `save_data()` — Save currently loaded expenses back to CSV

```python
def save_data():
    df_to_save = st.session_state.expenses.copy()
    # Convert datetime back to formatted string for saving
    df_to_save["Date"] = df_to_save["Date"].dt.strftime("%d-%m-%Y %I:%M %p")
    df_to_save.to_csv(EXPENSE_FILE, index=False)
```

- Copies the in-memory expense data from Streamlit session state.
- Formats the Date column as a string using the same pattern as when loading.
- Saves to `expenses.csv` without the index column.

### a. Chart Building Function

```python
def get_fig(df, ctype):
    if df.empty:
        return px.scatter(title="No data available for the selected period.")
    if ctype == "Donut":
        return px.pie(df, values="Amount", names="Category", hole=0.4, title="Spending by Category", color_discrete_sequence=CHART_COLORS)
    if ctype == "Bar":
        return px.bar(df, x="Category", y="Amount", color="Category", title="Spending by Category", color_discrete_sequence=CHART_COLORS)
    if ctype == "Line":
        return px.line(df.sort_values("Date"), x="Date", y="Amount", color="Category", title="Spending Over Time", color_discrete_sequence=CHART_COLORS)
    if ctype == "Treemap":
        return px.treemap(df, path=["Category", "Description"], values="Amount", title="Expense Treemap", color_discrete_sequence=CHART_COLORS)
    if ctype == "Box Plot":
        return px.box(df, x="Category", y="Amount", title="Spending Distribution by Category", color_discrete_sequence=CHART_COLORS)
    if ctype == "Sunburst":
        return px.sunburst(df, path=["Category", "Description"], values="Amount", title="Expense Sunburst", color_discrete_sequence=CHART_COLORS)
    return px.scatter(title="Unknown chart type")
```

- Returns an appropriate Plotly figure based on user selection.
- Uses different chart types to represent expense data visually:
  - Donut/Pie: % spending by category.
  - Bar chart: Amount per category.
  - Line chart: Spending over time per category.
  - Treemap and Sunburst: Hierarchical views by category and description.
  - Box plot: Distribution of spending per category.
